-  Bootstrap as CSS Framework
-  MySQL as Database Management

###################
First Configuration
###################

Set database conf in CI config.php file that match with your server 
and make sure that the read and write access to the project folder is enabled

Default:

-  hostname:	'localhost'
-  username:	'myproject'
-  password:	''
-  database:	'test_books'
-  dbdriver:	'mysqli'
